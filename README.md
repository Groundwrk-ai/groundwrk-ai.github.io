# groundwrk-ai.github.io

The GitHub user/organization Pages site for **Groundwrk-ai**, which claims the custom
domain `framewrklabs.com`. The root currently redirects to the Formwrk Material Planning
Tool at `framewrklabs.com/formwrk/` (the [`formwrk`](https://github.com/Groundwrk-ai/formwrk)
project repo).

Replace `index.html` with a real landing page whenever you want a proper root site.
